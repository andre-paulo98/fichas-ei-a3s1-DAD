This exercise populates a data table with content from the webservice

https://newsapi.org
to search for news from different sources.

Allow the user to select one news source from the ones returned from 

https://newsapi.org/v1/sources?country=us
The selected source is used to fetch the news from the endpoint 
https://newsapi.org/v1/articles?apiKey=6654c38654264552bfc3424d462aea2f&source=<source>

The returned object has a property "articles" with an array of objects. These objects have the following properties:

  • publishedAt
  • author
  • title
  • description
  • url
  • urlToImage

Note: Use a separate JavaScript code file "datatable.js"
